Software Requirements

  • Virtual Box or VMware or anyother virtual machine platform which supports VDI
  • Ubuntu 20.4 LTS image (find it here: VM)


Environment Setup

Since, we are trying to do Buffer Overflow (BoF) attack using Ubuntu, we need to turnoff a few countermeasures.

  • Address Space Randomization Ubuntu and other Linux-based systems use address space randomization, making it difficult to guess exact addresses, a critical step in buffer-overflow attacks, which can be disabled using the command: sudo sysctl -w kernel.randomize_va_space=0

  • Configuring /bin/sh In Ubuntu OS, the /bin/sh symbolic link points to the /bin/dash shell, which has a security countermeasure. This makes the attack more difficult as the victim program is a Set-UID program. To link /bin/sh to another shell without this countermeasure, we install zsh in a Ubuntu 20.04 VM: sudo ln -sf /bin/zsh /bin/sh
